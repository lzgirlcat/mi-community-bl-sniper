# mi-community-bl-sniper
A sniper for Bootloader Unlock Access for Xiaomi Devices that require permission from Xiaomi Community App

# "WHY would you use this when the XDA script exists???"
## the XDA `Avoid quota limit reached` script failed me twice in a row, misfiring before 00:00 Beijing Time
after checking out the awful code quality of it, i decided to make my own version to increase my chances. It worked **<u>first time</u>**

## Usage:
1. login to the mi community forum in your browser(s) of choice (the xda thread/script recommends using two tokens from 2 different browsers, but it worked fine with just one for me)
2. run `python -m sniper browser_name browser_2_name`, like `python -m sniper firefox` or `python -m sniper firefox chromium`
3. wait for midnight Beijing time
